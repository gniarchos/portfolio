01 TVTime - Shows Tracker

A TV Shows organizer to track and find information about any TV Show.

This project uses the Firebase Database for login/signup and for user data. The app fetches all the necessary data from two different APIs, TheMovieDB API and MDBList API.

FEATURES

- Track shows by adding them to your Watchlist

- Track episodes by marking them as watched or un-watched

- Mark entire seasons as watched

- Access complete cast and crew information

- Access details about seasons and shows

- Discover trending and popular shows that update daily

- View your history of watched episodes

- Search for any TV show and apply filters

- Track total watch time and episodes

- PWA (standalone app) for mobile devices

04 Wordle - Greek Edition

A Wordle clone in Greek language.

This project based on famous game Wordle.

How to play

The goal of the game is to guess the word within six tries. Each guess should be a valid five-letter word. After each guess the color of the tiles will change to indicate how close your guess was to the answer.

🟩 The letter is in the word and in the correct spot.

🟧 The letter is in the word but in the wrong spot.

⬛ The letter is not in the word in any spot.

FEATURES

- Daily infinite words

- User statistics

- Dark & Light mode

- Greek words only

- 7433 total words

- PWA (standalone app)

Live Demo Github Repository
